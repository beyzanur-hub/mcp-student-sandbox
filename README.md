# Mystery Module - Kuadratik Denklem Çözücü

## 📋 Genel Bakış
`mystery_module.py`, ikinci dereceden (kuadratik) denklemlerin köklerini belirlemek için tasarlanmış bir Python modülüdür.

## 🎯 Modülün Amacı
Bu modül, `ax² + bx + c = 0` formundaki denklemleri diskriminant yöntemiyle çözer.

## 📦 Yapılan Görevler
* **Refactoring:** `spaghetti_logic.py` temiz ve modüler hale getirildi.
* **Debugging:** `failing_calculator.py` hatası giderildi ve Pull Request açıldı.
* **Security:** `secret_leak.py` analiz edilerek AWS sızıntısı Issue olarak raporlandı.
* **Documentation:** Bu profesyonel README dosyası oluşturuldu.

## 🔧 Teknik Özellikler
* **Algoritma:** Quadratic Formula
* **Dil:** Python 3.x
